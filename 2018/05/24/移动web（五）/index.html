发表于 2018-05-24 | 分类于 移动web

今日css部分的一些不熟练的知识点：

  1. 两栏自适应时要注意浮动元素优先。
  2. box-shadow的参数问题：
    • 参数1：x轴的偏移
    • 参数2：y轴的偏移
    • 参数3：阴影的模糊度
    • 参数4：阴影的延伸
    • 参数5：阴影的颜色
    • 参数6：设置为inset时为内阴影
  3. -of-type选择器和-child选择器
    • 如p:first-of-type：找到p元素，通过p找到父元素，通过父元素找子元素中所有类型为p的元素，然后再去找到第一个p
    • 如p:first-child：找到p元素，通过p元素找到所有的子元素，找到第一个子元素，匹配判断类型（如果不是p那么就是无效选择器）

微金所项目总结：

  • 这次通过三天时间完成微金所项目，主要学习boostrap的基本使用方法，同时通过这个项目了解到了响应式开发原理和过程，掌握了插件使用的方法，复习了模板引擎artTemplate的使用，轮播图部分使用ajax完成了一个简单的交互功能。对boostrap中轮播图的改造使其满足我们的业务需求。
  • 本次项目中不足的地方：
    1. css代码的质量还不是特别好，应该学习boostrap中模块化开发的思维，提高代码的复用率。
    2. 需要提高开发的效率。
